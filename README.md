# Telecom Management System 📱

A web-based application built to streamline telecom operations, allowing administrators to manage users and enabling customers to view and purchase mobile recharge plans seamlessly.

---

## 🚀 Features

### 👤 Admin Panel
* **Secure Authentication:** Access control via dedicated admin username and password login.
* **User Management:** Register new customers directly into the system.
* **Data Overview:** View a comprehensive list of all registered users and their details.

### 💳 Customer & Recharge Services
* **Plan Browser:** View available mobile recharge plans (Prepaid/Postpaid).
* **Quick Recharge:** Perform instant mobile recharges by entering a mobile number and selecting a plan.

---
